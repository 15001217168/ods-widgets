odsSearchbox
directive in module ods-widgets

Description

This widget displays a wide searchbox that redirects the search on the Explore homepage of the domain.

Usage

as element:
<ods-searchbox
       placeholder="{string}"
       context="{CatalogContext}">
</ods-searchbox>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
placeholderstring

the text to display as a placeholder when the searchbox is empty

contextCatalogContext

Catalog Context indicating the domain to redirect the user to show the search results. If none, the search is done on the local domain (/explore/ of the current domain the user is).

(default: none)
