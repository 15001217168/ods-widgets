Welcome to Opendatasoft's widgets documentation!

Widgets are components that allow the integration of interactive elements (data visualizations, KPIs, etc.), from data available on an Opendatasoft API, directly on a web page.

Opendatasoft widgets are based on AngularJS. You will thus most likely encounter elements in the Opendatasoft widgets documentations that are directly inherited from AngularJS, but no prior knowledge of AngularJS is required to use the widgets.

Widgets can be used to build various types of web-based creations, such as:

  • a visualization (from the basic visualization to the visualization tool using several widgets) based on a public dataset, integrated into a blog article ;
  • an application, that allows users to design and build complex and interactive dashboards or visualization pages based on your data, using HTML tags ;
  • a website, either built on top of public data or your own data.

This documentation references all Opendatasoft widgets, filters and services. Each of those has its own reference page, which describes them, indicates their parameters and gives examples of their usage.

Use the menu on the left to navigate all available reference pages.

For more technical information about our widgets, please refer to the README of the Opendatasoft widgets GitHub repository.
