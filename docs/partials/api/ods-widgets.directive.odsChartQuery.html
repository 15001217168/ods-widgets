odsChartQuery
directive in module ods-widgets

Description

This widget can display a multiple chart generated using the "Charts" interface of OpenDataSoft.

Usage

as element:
<ods-chart-query
       context="{CatalogContext}"
       chart-config="{string|Object}">
</ods-chart-query>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContext

Dataset Context to use

chartConfigstringObject

A complete configuration, as a object or as a base64 string. The parameter directly expects an angular expression, so a base64 string needs to be quoted.

(default: none)
