odsTimerange
directive in module ods-widgets

Description

This widget displays two fields to select the two bounds of a date and time range.

Usage

as element:
<ods-timerange
       context="{DatasetContext}"
       time-field="{string}"
       default-from="{string}"
       default-to="{string}">
</ods-timerange>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContext

Dataset Context to use

timeFieldstring

Name of the field (date or datetime) to filter on

(default: first date/datetime field available)

defaultFromstring

Default datetime for the "from" field: either "yesterday" or "now"

(default: none)

defaultTostring

Default datetime for the "to" field: either "yesterday" or "now"

(default: none)
