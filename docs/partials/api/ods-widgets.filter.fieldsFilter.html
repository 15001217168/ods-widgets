fieldsFilter
filter in module ods-widgets

Usage

In HTML Template Binding

{{ fieldsFilter_expression | fieldsFilter:fields }}

In JavaScript

$filter('fieldsFilter')(fieldNames, fields)

Parameters

ParamTypeDetails
fieldNamesstring[]

A list of field names.

fieldsObject[]

A list of fields as returned by the API.

Returns

Object[]

A sublist of the fields input, containing only fields which are referenced in the fieldNames attribute.
