odsFilterSummary
directive in module ods-widgets

Description

This widget displays a summary of all the active filters on a context: text search, refinements...

Usage

as attribute
<ANY ods-filter-summary
     context="{CatalogContext|DatasetContext}"
     exclude="{string}"
     clear-all-button="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContextDatasetContext

Catalog Context or Dataset Context to display the filters of

excludestring

Optional: Name of parameters to not display, separated by commas. For example q,rows,start

(default: none)

clearAllButtonboolean

Optional: display a "clear all" button underneath the active filters' list.

(default: true)
