odsDatasetCard
directive in module ods-widgets

Description

If you wrap this directive around an element or a set of element, it will display an expandable card above it to show the title and description of the dataset, along with a link to the portal that shows the dataset, and the license attached to the data.

Usage

as element:
<ods-dataset-card
       context="{DatasetContext}">
</ods-dataset-card>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContext

Dataset Context to use

Example

<ods-dataset-context context="stations" stations-domain="public.opendatasoft.com" stations-dataset="jcdecaux_bike_data">
    <ods-dataset-card context="stations" style="height: 600px">
        <ods-map context="stations"></ods-map>
    </ods-dataset-card>
</ods-dataset-context>
