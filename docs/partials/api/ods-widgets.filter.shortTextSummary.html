shortTextSummary
filter in module ods-widgets

Usage

In HTML Template Binding

{{ shortTextSummary_expression | shortTextSummary:length }}

In JavaScript

$filter('shortTextSummary')(text, length)

Parameters

ParamTypeDetails
textstring

Some text

lengthnumber

The maximum length of the summary

Returns

string

A short summary from the given text, usually the first paragraph. If longer than the required length, an ellipsis will be made.
