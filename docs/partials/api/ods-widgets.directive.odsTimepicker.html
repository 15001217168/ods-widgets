odsTimepicker
directive in module ods-widgets

Description

This widget displays a field for the user to select a time (hours and minutes)

Usage

as element:
<ods-timepicker
       ng-model="{Number}"
       format="{string}"
       separators="{Array}"
       locale="{string}"
       minutes-interval="{Number}"
       force-interval="{Boolean}"
       placeholder="{string}"
       is-disabled="{Boolean}"
       is-readonly="{Boolean}">
</ods-timepicker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelNumber

The model this directive must use. It will be the number of minutes (hours * 60 + minutes)

formatstring

Force the timepicker to use a custom format. Every moment.js formats can be used if the formatted value has always the same length. It must be used with 'separators'

(default: none)

separatorsArray

An array of string representing the separators (or delimiters) in the specified format. It is used for caret manipulation

(default: none)

localestring

Force a specific locale to be used

(default: 'en')

minutesIntervalNumber

The minutes interval in the option list as an integer

(default: '5')

forceIntervalBoolean

If this is true, the user won't be able to type a time which does not respect the minutes interval

(default: false)

placeholderstring

The input field placeholder to use. By default, it displays the time format.

(default: format)

isDisabledBoolean

Specify if this time picker is disabled

(default: false)

isReadonlyBoolean

Specify if this time picker is readonly

(default: false)
