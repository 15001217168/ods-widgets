odsBreezometer
directive in module ods-widgets

Description

Integrates a Breezometer "widget" using the widget key provided by Breezometer

Usage

as element:
<ods-breezometer
       key="{string}"
       location="{string}">
</ods-breezometer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
keystring

The Breezometer Widget Key. See http://breezometer.com

locationstring

The city name.
