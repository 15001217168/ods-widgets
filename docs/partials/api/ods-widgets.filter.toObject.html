toObject
filter in module ods-widgets

Description

Transform an array of objects into an objet, using a key passed as a parameter.

Usage

In HTML Template Binding

{{ toObject_expression | toObject:key }}

In JavaScript

$filter('toObject')(array, key)

Parameters

ParamTypeDetails
arrayArray

An array of objects.

keyString

The key for the transformation.

Returns

Object

The array of objects converted into an object.

Example

[
  { "name": "foo", "count": 201 },
  { "name": "bar", "count": 202 }
]

mylist|toObject:'name'
{
  "foo": { "name": "foo", "count": 201 },
  "bar": { "name": "bar", "count": 202 }
}
