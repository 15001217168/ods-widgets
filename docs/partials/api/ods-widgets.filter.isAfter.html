isAfter
filter in module ods-widgets

Usage

In HTML Template Binding

In JavaScript

$filter('isAfter')(date1, date2)

Parameters

ParamTypeDetails
date1stringDateNumberArrayMoment

A date

date2stringDateNumberArrayMoment

Another date, which doesn't need to be in the same format as date1.

Returns

Boolean

Whether date1 is strictly after date2 or not, down to the millisecond.
