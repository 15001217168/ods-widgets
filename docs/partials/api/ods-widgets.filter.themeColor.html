themeColor
filter in module ods-widgets

Usage

In HTML Template Binding

{{ themeColor_expression | themeColor }}

In JavaScript

$filter('themeColor')(theme)

Parameters

ParamTypeDetails
themestring

A theme's slug (that is, its name normalized, see themeSlug)

Returns

string

The hexadecimal color code for this theme, as defined through ODSWidgetsConfig's theme setting.
