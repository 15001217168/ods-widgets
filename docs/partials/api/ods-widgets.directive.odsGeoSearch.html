odsGeoSearch
directive in module ods-widgets

Description

This widget displays a mini map with a draw-rectangle tool that can be used to search through a catalog.

Usage

as element:
<ods-geo-search
       context="{CatalogContext|CatalogContext[]}">
</ods-geo-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContextCatalogContext[]

Catalog context or array of contexts to use.
