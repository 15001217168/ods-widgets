odsAnalysis
directive in module ods-widgets

Description

This widget exposes the results of an analysis (as an object containing a results array and optionally an aggregations object) in a variable available in the scope. It can be used with AngularJS's ngRepeat to simply build a table of analysis results.

Usage

as attribute
<ANY ods-analysis="{string}"
     ods-analysis-context="{DatasetContext}"
     ods-analysis-max="{number}"
     ods-analysis-sort="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
odsAnalysisstring

Variable name to use

(default: analysis)

odsAnalysisContextDatasetContext

Catalog Context or Dataset Context to use

odsAnalysisMaxnumber

Maximum number of results to show

(default: all)

odsAnalysisSortstring

name of serie to sort on (or -serieName to invert the sort)
