odsChartSerie
directive in module ods-widgets

Description

odsChartSerie is the sub widget that defines a serie in the chart with all its parameters. see odsChart for complete examples.

Available chart types:

There are three available types of charts: simple series and areas that takes a minimal and a maximal value.

simple series

  • line
  • spline
  • area
  • areaspline
  • column
  • bar
  • pie
  • scatter

    areas

  • arearange
  • areasplinerange
  • columnrange

    available functions

  • COUNT
  • AVG
  • MIN
  • MAX
  • STDDEV
  • SUM
  • QUANTILES
  • CONSTANT

Usage

as element:
<ods-chart-serie
       [chart-type]="{string}"
       [function-y]="{string}"
       [expression-y]="{string}"
       [color]="{string}"
       [label-y]="{string}"
       [cumulative]="{boolean}"
       [logarithmic]="{boolean}"
       [min]="{integer}"
       [max]="{integer}"
       [display-units]="{boolean}"
       [multiplier]="{number}"
       [color-thresholds]="{string}"
       [subsets]="{string}"
       [subseries]="{boolean}"
       [refine-on-click-context]="{string}"
       [refine-on-click[context]-context-field]="{string}">
</ods-chart-serie>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[chartType]string

available types are: 'line', 'spline', 'arearange', 'areasplinerange', 'columnrange', 'area', 'areaspline', 'column', 'bar', 'pie', 'scatter'

[functionY]string

set up the function that will be used to calculate aggreation value. 'COUNT' counts the number of documents for each category defined by expressionY.

[expressionY]string

set up the facet used for aggregation

[color]string

defines the color used for this serie. see colors below

[labelY]string

specify a custom label for the serie

[cumulative]boolean

Y values are accumulated

[logarithmic]boolean

display the serie using a logarithmic scale

[min]integer

minimum value to be displayed on the Y axis

[max]integer

maximum value to be displayed on the Y axis

[displayUnits]boolean

enable the display of the units defined for the field in the tooltip

[multiplier]number

multiply all values for this serie by the defined number

[colorThresholds]string

an array of (value, color) objects. For each threshold value, if the Y value is above the threshold, the definedf color is used.

[subsets]string

used when functionY is set to 'QUANTILES' to define the wanted quantile

[subseries]boolean

an array containing 2 objects. TODO add explanation for this...

[refineOnClickContext]string

context name or array of of contexts name on which to refine when the serie is clicked on.

[refineOnClick[context]ContextField]string

name of the field that will be refined for each context.
