odsHubspotForm
directive in module ods-widgets

Description

Integrates a Hubspot form given a portal ID and the form ID.

Usage

as element:
<ods-hubspot-form
       portal-id="{string}"
       form-id="{string}">
</ods-hubspot-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
portalIdstring

The portal ID

formIdstring

The form ID
