ODSCurrentDomainProvider
service in module ods-widgets

Description

Use ODSCurrentDomainProvider to set configuration values for the current domain. The available settings are:

  • domainId - string - Value used as domain parameter for Catalog Contexts and Dataset Contexts when none is specified. Defaults is '' (empty string), which means a local API (root is /).

Example

var app = angular.module('ods-widgets').config(function(ODSCurrentDomainProvider) {
    ODSCurrentDomainProvider.setDomain('public');
});
