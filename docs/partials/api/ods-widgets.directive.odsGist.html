odsGist
directive in module ods-widgets

Description

Integrates a GitHub Gist widget into a page

Usage

as element:
<ods-gist
       username="{string}"
       id="{string}">
</ods-gist>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
usernamestring

The GitHub username

idstring

The Gist id. See the Gist URL to find it
