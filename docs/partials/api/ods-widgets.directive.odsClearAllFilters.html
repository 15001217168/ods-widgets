odsClearAllFilters
directive in module ods-widgets

Description

This widget displays a button which will clear all active filters in the given context.

Usage

as element:
<ods-clear-all-filters
       context="{CatalogContext|DatasetContext}">
</ods-clear-all-filters>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContextDatasetContext

Catalog Context or Dataset Context to display the filters of
