odsPicto
directive in module ods-widgets

Description

This widget displays a "picto" specified by a url and force a fill color on it. This element can be styled (height, width...), especially if the picto is vectorial (SVG).

Usage

as element:
<ods-picto
       url="{string}"
       color="{string}"
       classes="{string}">
</ods-picto>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
urlstring

The url of the svg or image to display

colorstring

The color to use to fill the svg

classesstring

The classes to directly apply to the svg element
