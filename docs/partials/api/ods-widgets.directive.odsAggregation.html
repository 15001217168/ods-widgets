odsAggregation
directive in module ods-widgets

Description

This widget exposes the results of an aggregation function over a context. Can be used for example to expose the average temperature of a weather dataset. The result is exposed into a new variable that you can use in other widgets or directly in your HTML.

Usage

as attribute
<ANY ods-aggregation="{string}"
     ods-aggregation-context="{CatalogContext|DatasetContext}"
     ods-aggregation-function="{string}"
     ods-aggregation-expression="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
odsAggregationstring

Variable name to use

(default: aggregation)

odsAggregationContextCatalogContextDatasetContext

Catalog Context or Dataset Context to use

odsAggregationFunctionstring

Aggregation function to apply (AVG, COUNT, MIN, MAX, STDDEV, SUM)

(default: COUNT)

odsAggregationExpressionstring

Expression to apply the function on, typically the name of a field. Optional only when the function is "COUNT".

(default: none)
