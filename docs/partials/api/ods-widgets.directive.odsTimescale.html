odsTimescale
directive in module ods-widgets

Description

Displays a control to select either: - last day - last week - last month - last year

Usage

as element:
<ods-timescale
       context="{DatasetContext}"
       time-field="{string}">
</ods-timescale>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContext

Dataset Context to use

timeFieldstring

Name of the field (date or datetime) to filter on
