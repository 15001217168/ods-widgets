odsAnalysisSerie
directive in module ods-widgets

Description

This widget exposes only keeps the longest serie in the results from an analysis. Results can be used as if coming from an analysis widget (and use a subaggregation on it for example)

Usage

as attribute
<ANY ods-analysis-serie="{string}"
     ods-analysis-serie-condition="{string}"
     ods-analysis-serie-name="{string}"
     ods-analysis-serie-separate-on-x="{string}"
     ods-analysis-serie-mode="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
odsAnalysisSeriestring

Analysis results

odsAnalysisSerieConditionstring

The condition to that the value must validate to be part of the serie. 'y' will be replaced by the value

odsAnalysisSerieNamestring

name of the serie to check for validation

odsAnalysisSerieSeparateOnXstring

name of the x axis in the analysis response used to split series

odsAnalysisSerieModestring

if mode is set to "reduce", keep only the longest serie of all splited series. Requires separate-on-x parameter.
