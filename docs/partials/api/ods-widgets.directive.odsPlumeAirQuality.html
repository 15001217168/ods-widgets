odsPlumeAirQuality
directive in module ods-widgets

Description

Integrates a Plume Air Embed using a city name.

Usage

as element:
<ods-plume-air-quality
       city="{string}"
       lang="{string}">
</ods-plume-air-quality>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
citystring

The name of the city you want to integrate. See http://www.plumelabs.com/embed/ for more information.

langstring

fr_fr for the french version, en_us for the english one.
