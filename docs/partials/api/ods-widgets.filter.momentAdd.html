momentadd
filter in module ods-widgets

Usage

In HTML Template Binding

{{ momentadd_expression | momentadd:precision:number }}

In JavaScript

$filter('momentadd')(date, precision, number)

Parameters

ParamTypeDetails
datestringDateNumberArrayMoment

A date

precisionstring

A unit describing the type of the number parameter. Can be any of years, quarters, months, weeks, days, hours, minutes, seconds or milliseconds.

numbernumber

How many years, hours, minutes (depending on precision) should be added. Can be a negative number.

Returns

Moment

A date
