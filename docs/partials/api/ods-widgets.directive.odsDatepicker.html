odsDatepicker
directive in module ods-widgets

Description

This widget displays a field for the user to select a date

Usage

as element:
<ods-datepicker
       ng-model="{Number}"
       format="{string}"
       separators="{Array}"
       locale="{string}"
       enable-time="{Boolean}"
       placeholder="{string}"
       is-disabled="{Boolean}"
       is-readonly="{Boolean}"
       disabled-dates="{Array}"
       min-date="{Date}"
       max-date="{Date}">
</ods-datepicker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelNumber

The model this directive must use. It will be a javascript date

formatstring

Force the datepicker to use a custom format. Every moment.js formats can be used if the formatted value has always the same length. It must be used with 'separators'

(default: none)

separatorsArray

An array of string representing the separators (or delimiters) in the specified format. It is used for caret manipulation

(default: none)

localestring

Force a specific locale to be used

(default: 'en')

enableTimeBoolean

Display the time picker for hours/minutes selection

(default: false)

placeholderstring

The input field placeholder to use. By default, it displays the date format.

(default: format)

isDisabledBoolean

Specify if this date picker is disabled

(default: false)

isReadonlyBoolean

Specify if this date picker is readonly

(default: false)

disabledDatesArray

An array of javascript date which will be disabled for selection

(default: none)

minDateDate

The minimum date available for selection

(default: none)

maxDateDate

The maximum date available for selection

(default: none)
