odsTextSearch
directive in module ods-widgets

Description

This widget displays a search box that can be used to do a full-text search on a context.

Usage

as element:
<ods-text-search
       placeholder="{string}"
       button="{string}"
       context="{CatalogContext|DatasetContext}">
</ods-text-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
placeholderstring

the text to display as a placeholder when the searchbox is empty

buttonstring

the text to display in the "search" button

contextCatalogContextDatasetContext

Catalog Context or Dataset Context to use
