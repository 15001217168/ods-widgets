odsTextSearch
directive in module ods-widgets

Description

This widget displays a search box that can be used to do a full-text search on a context.

Usage

as element:
<ods-text-search
       placeholder="{string}"
       show-button="{string}"
       button="{string}"
       field="{string}"
       context="{CatalogContext|DatasetContext}">
</ods-text-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
placeholderstring

the text to display as a placeholder when the searchbox is empty

showButtonstring

display the "search" button

(default: false)

buttonstring

the text to display in the "search" button

fieldstring

The name of a field you want to restrict the search on (i.e. only search on the textual content of a specific field). The search will be a simple text search and won't support any query language or operators.

(default: none)

contextCatalogContextDatasetContext

Catalog Context or Dataset Context to use
